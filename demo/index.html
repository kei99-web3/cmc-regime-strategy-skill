CMC Regime Strategy Skill

BNB HACK Track 2 concept: a CMC-native skill that turns market data into a backtestable, risk-bounded strategy spec. No live trading, no wallet, no transaction.

Track 2: Strategy Skills CMC Agent Hub / Data MCP ready Backtestable spec Local fixture demo

Judge Flow

1. User asks for a BNB ecosystem strategy for the next 7 days.

2. The skill reads CMC-style quotes, technicals, derivatives, social/news, and market context.

3. It classifies the regime and emits a strategy spec with entry, exit, risk, invalidation, and backtest instructions.

4. It stops before Trust Wallet, BNB execution, x402 payment, or any transaction.
